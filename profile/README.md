# Vibora Ink Calculadora 

Projeto Integrador V de Engenharia de Software desenvolvido para criar um sistema web de orçamento de tatuagens com apoio de Inteligência Artificial.

## Visão geral

A proposta do projeto é permitir que o cliente descreva a tatuagem desejada por meio de um formulário. Após o envio, esses dados são processados no back-end, onde a IA analisa as informações e determina um peso de dificuldade para a tatuagem.

Com base nesse resultado, o sistema gera uma segunda etapa para a tatuadora preencher os gastos envolvidos no serviço. Ao final, o orçamento é criado a partir da dificuldade estimada pela IA e dos custos definidos pela profissional.

## Objetivo

Desenvolver um sistema web inteligente para cálculo e estimativa de custos de tatuagens, automatizando parte do processo de orçamento e organizando melhor a interação entre cliente, back-end, IA e tatuadora.

## Objetivos específicos

- Coletar informações detalhadas da tatuagem por meio de formulário.
- Enviar os dados do cliente para processamento no back-end.
- Utilizar IA para determinar o peso de dificuldade da tatuagem.
- Criar uma segunda etapa de preenchimento para a tatuadora informar os gastos.
- Gerar o orçamento final com base na dificuldade e nos custos informados.
- Tornar o processo de orçamento mais claro, estruturado e padronizado.

## Fluxo resumido

1. O cliente preenche o formulário com os dados da tatuagem.
2. O formulário é enviado ao back-end.
3. A IA analisa as informações e define o peso de dificuldade da tatuagem.
4. O sistema gera um segundo formulário para a tatuadora.
5. A tatuadora informa os gastos e valores necessários.
6. O orçamento final é criado.

## Tecnologias previstas

De acordo com a documentação, a solução considera o uso das seguintes tecnologias:

- `React` para a interface.
- `HTML5` para a estrutura da aplicação web.
- `Tailwind CSS` para estilização responsiva.
- `Python` para API e integração com modelos.
- `MySQL` para armazenamento dos pedidos.
- `Llama` para análise textual em português.

## Diferenciais da solução

Em comparação com soluções de mercado citadas na documentação, o projeto busca combinar:

- formulário estruturado;
- uso de IA para avaliar a dificuldade da tatuagem;
- participação da tatuadora na definição dos gastos;
- consideração de múltiplas variáveis de negócio;
- geração de orçamento detalhado em duas etapas.
